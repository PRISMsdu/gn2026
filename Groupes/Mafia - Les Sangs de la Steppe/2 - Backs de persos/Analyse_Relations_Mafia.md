## Analyse des Relations

### Hiérarchie
- **Kaelan** : Chef suprême, relations d'autorité avec tous
- **Vorak** : Second, coordination opérationnelle
- **Autres membres** : Spécialistes avec relations de collaboration

### Relations Clés
- **Kaelan ↔ Vorak** : Confiance absolue, relation de commandement
- **Kaelan ↔ Shadow** : Confiance limitée, surveillance nécessaire
- **Selena ↔ Lyanna** : Alliance stratégique pour les relations extérieures
- **Drask ↔ Gareth** : Support médical et fourniture de poisons
- **Shadow ↔ Raven** : Partage d'informations et coordination

### Types de Relations
- **Autorité** : Kaelan sur tous, Vorak sur les opérationnels
- **Collaboration** : Entre spécialistes complémentaires
- **Support** : Services mutuels (médical, protection, information)
- **Surveillance** : Kaelan sur Shadow (confiance limitée)

### Dynamiques Internes

#### Relations de Confiance
- **Haute confiance** : Kaelan-Vorak, Kaelan-Raven
- **Confiance limitée** : Kaelan-Shadow (surveillance)
- **Confiance conditionnelle** : Autres membres entre eux

#### Alliances Stratégiques
- **Selena-Lyanna** : Alliance pour les relations extérieures
- **Drask-Gareth** : Alliance médicale et opérationnelle
- **Shadow-Raven** : Alliance informationnelle

#### Points de Tension Potentiels
- **Shadow** : Confiance limitée de Kaelan, surveillance constante
- **Relations commerciales** : Lyanna en interface avec l'extérieur
- **Secrets médicaux** : Drask détient des informations sensibles

### Implications pour le Jeu

#### Scénarios Possibles
- **Trahison de Shadow** : Kaelan pourrait l'éliminer
- **Alliance Selena-Lyanna** : Pourrait contourner l'autorité
- **Support Drask-Gareth** : Indispensable pour les opérations

#### Hooks d'Intrigue
- **Révélation des secrets** : Drask pourrait révéler des informations
- **Surveillance de Shadow** : Découverte d'activités suspectes
- **Négociations extérieures** : Lyanna pourrait trahir
