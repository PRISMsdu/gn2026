# Groupe Banquiers - UBI

## Structure des Répertoires

### 1 - Back de groupe
Contient les fichiers de background du groupe :
- `Back_groupe_UBI_Orga.md` - Background principal du groupe
- `Charte_UBI.md` - Charte et règles du groupe
- `Back_groupe_UBI.md` - Historique du groupe

### 2 - Backs de persos
Contient les backgrounds des personnages (étape de création) :
- `perso_UBI_*-orga.md` - Backgrounds des personnages
- `Analyse_Relations_Banquiers.md` - Analyse des relations
- `Diagramme_Relations_Banquiers.md` - Diagramme des relations

### 3 - Roles des Joueurs
Contient les rôles finaux des joueurs (générés à partir des backgrounds).

## Workflow
1. **Intrigue** → Définit les objectifs et rôles nécessaires
2. **Back de groupe** → Structure et histoire du groupe
3. **Backs de persos** → Backgrounds des personnages (fichiers -orga)
4. **Roles des Joueurs** → Rôles finaux distribués aux joueurs
